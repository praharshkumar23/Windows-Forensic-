## üìÇ Investigating Windows Event Logs

### What Are Windows Event Logs?
Windows Event Logs record:
- User logins and logouts
- Authentication failures
- System-level changes
- Application crashes
- Security policy modifications

### Key Log Sources
- **Security**
- **System**
- **Application**
